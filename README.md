# DishCovery

A restaurant review application built with Angular 21 and Angular Material.

## Quick Start

```bash
yarn install
yarn server    # Start mock API (port 3000)
yarn start     # Start dev server (port 4200)
```

## Tech Stack

| Technology | Purpose |
|------------|---------|
| Angular 21 | Frontend framework (standalone components) |
| Angular Material | UI component library |
| RxJS | Reactive state management |
| JSON Server | Mock REST API |
| Karma + Jasmine | Unit testing |
| ESLint | Linting |
| Husky + commitlint | Commit conventions |
